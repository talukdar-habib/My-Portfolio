Vanilla JavaScript

Vanilla JavaScript refers to using plain JavaScript without any libraries or frameworks. It's the pure, standard version of JavaScript, allowing developers to manipulate the DOM, handle events, and interact with APIs directly.

React

React is a popular JavaScript library developed by Facebook for building user interfaces, especially single-page applications. It allows developers to create reusable UI components, making code more modular and easier to manage.

Node.js

Node.js is a runtime environment that allows you to run JavaScript on the server side. It uses the V8 engine from Google Chrome to execute code, making it fast and efficient.

MongoDB

MongoDB is a NoSQL database that stores data in flexible, JSON-like documents instead of traditional rows and columns. It is designed to handle large volumes of unstructured or semi-structured data, making it ideal for modern applications with evolving data needs.
